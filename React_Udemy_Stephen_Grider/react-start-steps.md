# Steps to create React Project using VITE

<!-- creating a directory -->
- mkdir `react-vite-app`
<!-- moving into the created react project -->
- cd react-vite-app
<!-- Initialize the Vite project: -->
- npm create vite@latest .
- npm install
- npm run dev
- npm run build
- npm run preview
